Project 2: Advanced Data, APIs, & Sensor Integration

Problem Statement: A modern health app needs to interact with the real world. By adding hardware sensors, cloud backups, and live internet data, the app becomes a fully-fledged tracking ecosystem.

Features Implemented:

  • Web API: Fetching daily health tips using Retrofit.
  • Hardware Sensor: Tracking movement via the device's Accelerometer.
  • Local Persistence: Saving steps permanently via Room.
  • Cloud Integration: Backing up user data to Firebase Firestore.

Lessons Learned: Successfully integrated complex external libraries and hardware APIs without freezing the main UI thread. Understanding Coroutines was crucial here.
